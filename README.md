# Job Tracker

> Because "I'm pretty sure I applied there already." is not a tracking system.

## Why?

After enough job applications, every company name starts looking familiar.

You know you've seen that logo before.

You remember writing a cover letter.

You just can't remember whether you actually clicked **Submit**.

So instead of trusting my memory, I decided to trust PostgreSQL.

## What is it?

Job Tracker is a personal application management system.

It keeps track of where you've applied, what position it was for, when you applied, which version of your CV you sent, and where to check the application status later.

The goal isn't to help you find jobs.

It's to help you remember everything after you've found them.

## Planned Features

- Create and manage job applications
- Track application status
- Save company and job links
- Store the application portal URL
- Record notes and follow-up reminders
- Dashboard with application statistics
- Search and filtering
- Authentication

## Design Philosophy

The project isn't trying to replace LinkedIn, Indeed, or an Applicant Tracking System.

It's solving a much smaller problem:

"I've applied to 63 jobs. I have absolutely no idea what's happening anymore."

## Disclaimer

This application can help you stay organised.

It cannot make recruiters reply to your emails.

I'm still researching that feature.
